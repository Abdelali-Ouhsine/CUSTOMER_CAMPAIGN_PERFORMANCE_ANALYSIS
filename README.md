📊 MARKETING CAMPAIGN PERFORMANCE ANALYSIS
🧾 Project Overview
This project focuses on analyzing marketing campaign effectiveness and identifying customer purchasing behavior. The goal is to transform raw data into actionable insights using Python and Power BI.
🎯 Objectives
Perform Exploratory Data Analysis (EDA)
Analyze the impact of marketing campaigns on sales
Identify top-performing products
Build an interactive Power BI dashboard for decision-making
🛠️ Tech Stack
Python: pandas, numpy, scipy, statsmodels
Visualization: matplotlib, seaborn, plotly
Power BI: Power Query, Data Modeling, DAX
Tools: Git, GitHub, Jira
📂 Project Structure
├── data/                # Raw dataset
├── notebooks/           # EDA and statistical analysis
├── powerbi/             # Power BI file (.pbix)
├── images/              # Dashboard screenshots
└── README.md
🔍 Key Steps
1. Exploratory Data Analysis (EDA)
Data cleaning (missing values, duplicates)
Variable type identification
Descriptive statistics
Visualizations (histograms, boxplots, bar charts)
2. Statistical Analysis
t-test / ANOVA for sales comparison
Chi-square test for categorical variables
Correlation analysis
Interpretation of p-values
3. Data Transformation
Pivot/Unpivot tables (campaigns, products, platforms)
Data cleaning and formatting in Power Query
4. Data Modeling
Relationship creation using Customer ID
DAX measures (sales, campaign performance)
5. Dashboard Development
Campaign performance visuals
Product sales analysis
Platform distribution
Interactive filters and slicers
📈 Dashboard Features
Revenue per campaign and product
Number of customers per campaign
Sales breakdown by product
Purchase distribution by platform
Dynamic filtering for analysis
🚀 Key Insights
Identified the most effective marketing campaigns
Highlighted top-selling products
Revealed customer behavior patterns across platforms
